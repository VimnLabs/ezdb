## Get the registry
Get the log contents of your backups, they come from the `index.json` from where you set the folder to store your backups.
```ts copy wrap twoslash showLineNumbers
import { Backup } from "@vimn/ezdb"
const backup = new Backup({
	backups: './backups', // Folder to save backups
	path: './', // Path to be backed up, can be a single file or a folder
	heartbeat: 30 * 60 * 1000, // Milliseconds between each backup, in this example it is 30 minutes.

	verbose: {
		start: true,
		open_folder: false,
		open_file: false,
		end: true,
		use: false
	}
});
// ---cut---
backup.registry
```

```ts copy wrap twoslash showLineNumbers
	backups: './backups', // Folder to save backups

	verbose: {
		start: true,
		open_folder: false,
		open_file: false,
		end: true,
		use: false
	}

// ---cut---
backup.create()
```

## Get a backup
Search a backup, returns a [backup info](/ezdb/reference/type-aliases/backupinfo/) or `null`
```ts copy wrap twoslash showLineNumbers
import { Backup } from "@vimn/ezdb"
const backup = new Backup({
	backups: './backups', // Folder to save backups
	path: './', // Path to be backed up, can be a single file or a folder
	heartbeat: 30 * 60 * 1000, // Milliseconds between each backup, in this example it is 30 minutes.

	verbose: {
		start: true,
		open_folder: false,
		open_file: false,
		end: true,
		use: false
	}
});

//@ts-expect-error
// ---cut---
backup.info(/* SEARCH */) // This can be the id (timesteamp) or name of the zip (for example: dpiitqkszwe4rri8).
```
