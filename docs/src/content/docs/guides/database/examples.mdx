## Utility functions
### Database Ping 
Get the ping of the database
```ts copy wrap twoslash showLineNumbers
import { Database } from "@vimn/ezdb"
const AUTH_KEY = process.env.DATABASE_AUTH as string
const database = new Database({
	root: './database/',
	tables: ['main'],

	auth: AUTH_KEY
});
// ---cut---
database.ping()
```
<LinkCard title="Reference" href="/ezdb/reference/classes/database/#ping" />

### Sanitize string
Example content for the ‚Äúmain‚Äù table
```json title="main.json"
{
	"foo": {
		"bar": "üë∏üèª You found me Mario!",
	},
	"foo.bar" : "üçÑ The princess is in another castle!",
}
```

```ts copy wrap twoslash showLineNumbers
import { Database } from "@vimn/ezdb"
const AUTH_KEY = process.env.DATABASE_AUTH as string
const database = new Database({
	root: './database/',
	tables: ['main'],

	auth: AUTH_KEY
});
// ---cut---
// This returns "üçÑ The princess is in another castle!"
database.get({
  table: "main",
  key: database.sanitize("foo.bar"),
  auth: AUTH_KEY
})

// This returns "üë∏üèª You found me Mario!"
database.get({
  table: "main",
  key: "foo.bar",
  auth: AUTH_KEY
})
```

## Basic usages
### Get function
```ts copy wrap twoslash showLineNumbers
import { Database } from "@vimn/ezdb"
const AUTH_KEY = process.env.DATABASE_AUTH as string
const database = new Database({
	root: './database/',
	tables: ['main'],

	auth: AUTH_KEY
});
// ---cut---
// Get an entire table
database.get({
  table: "main",
  auth: AUTH_KEY
})
// Get a specific value of a key in the table, or get the entire table
database.get({
  table: "main",
  key: "foo",
  auth: AUTH_KEY
})
```
<LinkCard title="Reference" href="/ezdb/reference/classes/database/#get" />
### Set function
```ts copy wrap twoslash showLineNumbers
import { Database } from "@vimn/ezdb"
const AUTH_KEY = process.env.DATABASE_AUTH as string
const database = new Database({
	root: './database/',
	tables: ['main'],

	auth: AUTH_KEY
});
// ---cut---
// Overwrite a table completely
database.set({
  table: "main",
  auth: AUTH_KEY,
  value: {foo: "bar"}
})
// Writes a specific property
database.set({
  table: "main",
  key: "foo",
  value: "bar",
  auth: AUTH_KEY
})
```
<LinkCard title="Reference" href="/ezdb/reference/classes/database/#set" />

### Multi function
```ts copy wrap twoslash showLineNumbers

const database = new Database({
	root: './database/',
	tables: ['main'],

	auth: AUTH_KEY
});
// ---cut---
database.multi({
  table: "main",
  key: "foo",
  value: 2, //THIS SHOULD ALWAYS BE A NUMBER
  auth: AUTH_KEY
})
```
<LinkCard title="Reference" href="/ezdb/reference/classes/database/#multi" />

<Aside type="note">
The value given will be the number to multiply the one already in the table, for example, if foo is worth 2, and in value we give it 2, it would be 2 times 2, then it would be overwritten as 4.
</Aside>
